# 🔍 InfoHunter

**InfoHunter** is a Python-based GUI reconnaissance tool designed for information gathering, network analysis, and file metadata extraction.

It provides a simple interface built with Tkinter to perform multiple security and OSINT tasks in one tool.

---

## 🚀 Features

- 🌐 DNS Lookup (IPv4 / IPv6)
- 🧾 WHOIS Information Retrieval
- 🔎 DNS Records Extraction (A, MX, TXT, NS, AAAA)
- 🚪 Port Scanner (Fast multi-threaded scanning)
- 🌍 Subdomain Enumeration
- 🖥️ Website Information Gathering
- 📂 Directory Brute Force Scanner
- 🖼️ Image Metadata Analysis (EXIF data)
- 📄 PDF Metadata Extraction
- 📊 Export results to CSV

---

## 🖥️ Interface

Built with:
- Python Tkinter GUI
- TreeView results display
- Live console logs
